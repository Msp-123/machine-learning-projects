1. What is this report?

This report explains the global behavior of the XGBoost fraud detection model using SHAP. SHAP values show how much each feature contributes to increasing or decreasing the model output.

SHAP explanations are especially useful in fraud detection because business users need to understand why a claim is considered risky, not only whether it is classified as fraud.
